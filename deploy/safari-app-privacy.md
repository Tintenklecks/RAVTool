# Safari / App Store Privacy

Use this as the starting point for App Store Connect privacy answers.

## Data collection

JobRoom Helper for Safari does not collect data from the user or device and does not send job data to the developer or a third party.

## Local-only data

The Safari extension stores the last job read from LinkedIn in local Safari extension storage so the user can paste it into Job-Room. This data remains local to the user's browser.

## Tracking

No tracking.

## Analytics

No analytics.

## Advertising

No advertising.

## External accounts

The extension does not create an account. LinkedIn and Job-Room accounts, if used, are handled by those websites directly.
